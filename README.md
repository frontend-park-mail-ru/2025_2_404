## Скрипты

В проекте настроены команды для проверки и автоисправления кода.
(код находится в package.json("scripts")).
Желательно поставить себе следующие расширения в vscode(ESlint(microsoft), Stylelint(stylelint), 
Prettier - Code formatter), тогда ошибки будут отображаться в лайве
### Линтинг JavaScript
- `npm run lint:js` — проверка всех JS файлов через **ESLint**.
- `npm run lint:js:fix` — проверка и автоматическое исправление проблем.

### Линтинг CSS
- `npm run lint:css` — проверка всех CSS файлов через **Stylelint**.
- `npm run lint:css:fix` — проверка и автоматическое исправление проблем.

### Форматирование
- `npm run format` — форматирование всех файлов в проекте через **Prettier**.

### Общие команды
- `npm run lint` — запустить все проверки (JS + CSS).
- `npm run lint:fix` — исправить все возможные ошибки (JS + CSS) и отформатировать проект.
